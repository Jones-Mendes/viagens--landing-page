# TripMe - Landing Page de Viagens

Este projeto e uma landing page para um site de viagens. O objetivo e apresentar a marca, destacar servicos e incentivar o visitante a iniciar uma conversa ou solicitar um roteiro personalizado.

## Estrutura do projeto

- index.html: markup da landing page
- styles.css: estilos principais
- image/europa.avif: imagem de destaque da hero
- image/globo.gif: favicon

## Como visualizar

Abra o arquivo index.html no navegador.

## Tecnologias

- HTML5
- CSS3
